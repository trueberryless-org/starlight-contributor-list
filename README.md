## Package

If you are looking for the Astro package, you can find it in the [`packages/astro-contributors/`](/packages/astro-contributors/) directory.

## Project structure

This project uses pnpm workspaces to develop a single Astro component package from the `packages/astro-contributors/` directory. A Starlight documentation site is also available in the `docs/` directory that is also used for testing and demonstrating the Astro component.
