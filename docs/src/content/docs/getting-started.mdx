---
title: Getting Started
---

Display a list of all contributors to your project.

## Prerequisites

You will need to have an Astro website set up.
If you don't have one yet, you can follow the ["Install and Setup"](https://docs.astro.build/en/install-and-setup/) guide in the Astro docs to create one.

:::note
If your repository is **private**, you need to configure the `PUBLIC_GITHUB_TOKEN` environment variable in your [GitHub Secrets](https://docs.github.com/en/actions/security-for-github-actions/security-guides/using-secrets-in-github-actions). If the repository is public, you can skip this step.
:::

## Installation

import { Steps, Tabs, TabItem } from "@astrojs/starlight/components";

<Steps>

1.  `astro-contributors` is an Astro [component](https://docs.astro.build/en/basics/astro-components/). Install the package by running the following command in your terminal:

    <Tabs syncKey="pkg">

    <TabItem label="npm">

    ```sh
    npm install astro-contributors
    ```

    </TabItem>

    <TabItem label="pnpm">

    ```sh
    pnpm add astro-contributors
    ```

    </TabItem>

    <TabItem label="Yarn">

    ```sh
    yarn add astro-contributors
    ```

    </TabItem>

    </Tabs>

2.  In any `.mdx` or `.astro` file, import the `ContributorList` component by adding the following import statement:

    <Tabs syncKey="file">

    <TabItem label="mdx">

    ```mdx
    import { ContributorList } from "astro-contributors";
    ```

    </TabItem>

    <TabItem label="astro">

    ```astro
    ---
    import { ContributorList } from "astro-contributors"
    ---
    ```

    </TabItem>

    </Tabs>

3.  Now you can use the `ContributorList` component within this file:

    <Tabs syncKey="file">

    <TabItem label="mdx">

    ```mdx
    <ContributorList githubRepo="trueberryless-org/mutanuq" />
    ```

    </TabItem>

    <TabItem label="astro">

    ```astro
    <ContributorList githubRepo="trueberryless-org/mutanuq" />
    ```

    </TabItem>

    </Tabs>

4.  [Start the development server](https://docs.astro.build/en/develop-and-build/#start-the-astro-dev-server) to preview the plugin in action.

</Steps>

Check out the [parameters](/parameters) which you can pass to the `ContributorList` component.
