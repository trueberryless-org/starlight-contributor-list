---
title: Getting Started
---

Display a list of all contributors to your project.

## Prerequisites

You will need to have a Starlight website set up.
If you don't have one yet, you can follow the ["Getting Started"](https://starlight.astro.build/getting-started) guide in the Starlight docs to create one.

## Installation

import { Steps, Tabs, TabItem } from "@astrojs/starlight/components";

<Steps>

1. Configure the `PUBLIC_GITHUB_TOKEN` environment variable in your [GitHub Secrets](https://docs.github.com/en/actions/security-for-github-actions/security-guides/using-secrets-in-github-actions).

1. `starlight-contributor-list` is a Starlight [plugin](https://starlight.astro.build/reference/plugins/). Install it by running the following command in your terminal:

    <Tabs syncKey="pkg">

    <TabItem label="npm">

    ```sh
    npm install starlight-contributor-list
    ```

    </TabItem>

    <TabItem label="pnpm">

    ```sh
    pnpm add starlight-contributor-list
    ```

    </TabItem>

    <TabItem label="Yarn">

    ```sh
    yarn add starlight-contributor-list
    ```

    </TabItem>

    </Tabs>

1. In any `.mdx` or `.astro` file, import the `ContributorList` component by adding the following import statement:

    <Tabs syncKey="file">

    <TabItem label="mdx">

    ```mdx
    import { ContributorList } from "starlight-contributor-list";
    ```

    </TabItem>

    <TabItem label="astro">

    ```astro
    ---
    import { ContributorList } from "starlight-contributor-list"
    ---
    ```

    </TabItem>

    </Tabs>

1. Now you can use the `ContributorList` component within this file:

    <Tabs syncKey="file">

    <TabItem label="mdx">

    ```mdx
    <ContributorList githubRepo="trueberryless-org/mutanuq" />
    ```

    </TabItem>

    <TabItem label="astro">

    ```astro
    <ContributorList githubRepo="trueberryless-org/mutanuq" />
    ```

    </TabItem>

    </Tabs>

1. [Start the development server](https://starlight.astro.build/getting-started/#start-the-development-server) to preview the plugin in action.

</Steps>
